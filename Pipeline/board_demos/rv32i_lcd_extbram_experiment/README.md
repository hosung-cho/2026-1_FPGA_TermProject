# RV32I LCD External BRAM Experiment

This is the separate, runtime-loadable data-memory version of the verified RV32I + LCD demo.

Use this when multiple input images need to be loaded from the PS side without rebuilding the RV32I hardware each time.

Start with:

```text
docs/RV32I_LCD_EXTBRAM_RUNBOOK.md
```

Verified board result:

```text
EXTBRAM_PRED=7
EXTBRAM_EXPECTED=7
EXTBRAM_PASS=1
```
