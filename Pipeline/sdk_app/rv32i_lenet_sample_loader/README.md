# RV32I LeNet Sample Loader SDK App

This app targets the sample-load PS/PL design generated by:

```powershell
cd Pipeline\vivado_board
& 'C:\Xilinx\Vivado\2019.1\bin\vivado.bat' -mode batch -source .\run_ps_loadable_bd_impl.tcl
```

It keeps the LeNet firmware and weights fixed in PL memory, then lets the PS
replace only the 32x32 INT8 input image and expected label before each run.

Generate the sample header before importing the app into SDK:

```powershell
cd Pipeline\sdk_app\rv32i_lenet_sample_loader
python .\make_lenet_samples_header.py
```

AXI map from base `0x43C00000`:

```text
0x0000 control/status registers
0x1000 input image words, 256 x 32-bit = 1024 bytes
0x2000 expected label word
```

Expected UART summary:

```text
summary pass=10 total=10
```
