# RV32I LeNet Board Build

Target FPGA part: `xc7z020clg484-1`.

This directory is for a real FPGA board build. The part number is enough for
Vivado device selection, but it is not enough for a safe bitstream. You still
need the board's real pin constraints.

## Files

- `run_board_impl.tcl`: Vivado batch build for `RV32I_Board_Top`.
- `board_template.xdc`: pin constraint template. Copy it to `board.xdc`.
- Generated by the Tcl script:
  - `imem.hex`: copied from `FPGA_proj/firmware/lenet_infer_imem.hex`
  - `lenet_digit7_dmem.mem`: copied from `FPGA_proj/firmware/lenet_digit7_dmem.mem`
  - `build/rv32i_lenet_board.bit`

## LED Mapping

- `LED[0]`: LeNet firmware wrote done flag.
- `LED[4:1]`: predicted digit.
- `LED[5]`: pass, predicted digit is 7 for the bundled digit7 input.
- `LED[6]`: fail, predicted digit is not 7.
- `LED[7]`: heartbeat.

## Run

1. Copy `board_template.xdc` to `board.xdc`.
2. Fill in the real clock, reset, and LED package pins for your board.
3. Run:

```powershell
cd Pipeline\vivado_board
& 'C:\Xilinx\Vivado\2019.1\bin\vivado.bat' -mode batch -source .\run_board_impl.tcl
```
